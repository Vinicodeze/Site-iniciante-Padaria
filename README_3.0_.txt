Padaria Pão de Mel

Bem-vindo ao site da Padaria Pão de Mel! Este é um projeto de exemplo de um site de padaria feito com HTML e CSS, que apresenta nossos produtos, galeria, vídeo institucional e informações de localização, além de formulários de pedidos, feedback e cadastro de clientes.

Sobre o Projeto

O site foi desenvolvido para mostrar:

Quem somos e nossa missão.

Nossos produtos (pães, doces, salgados, bebidas e sanduíches).

Cardápio completo com preços.

Galeria de imagens da padaria e equipe.

Vídeo mostrando nosso dia a dia.

Localização e horários de funcionamento.

Formulário de pedidos especiais.

Formulário de feedback dos clientes.

Formulário de cadastro de clientes.

Tecnologias Utilizadas

HTML5

CSS básico
